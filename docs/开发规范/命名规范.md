# 命名规范

## 文件命名

| 类型 | 命名规则 | 示例 |
|------|---------|------|
| Controller | `{Model}Controller.php` | `CompanyController.php` |
| Model | `{Model}.php` (单数) | `Company.php` |
| Request | `{Model}{Action}Request.php` | `CompanyStoreRequest.php` |
| Resource | `{Model}Resource.php` | `CompanyResource.php` |
| Filter | `{Model}Filter.php` | `CompanyFilter.php` |
| Service | `{Model}Service.php` | `CompanyService.php` |

## 数据库表命名

**格式**: `{module_prefix}_{table_name}` (复数, snake_case)

| 模块 | 表名 |
|------|------|
| Mdm | `mdm_companies`, `mdm_company_operators` |
| ScanInvoice | `scan_invoice_records` |
| Iam | `iam_users`, `iam_roles`, `iam_permissions` |

## 目录结构

```
Modules/{Module}/
├── app/
│   ├── Http/
│   │   ├── Controllers/{Model}Controller.php
│   │   ├── Requests/{Model}/{Model}StoreRequest.php
│   │   └── Resources/{Model}Resource.php
│   ├── Models/{Model}.php
│   ├── ModelFilters/{Model}Filter.php
│   └── Services/{Model}Service.php
├── database/migrations/
└── routes/api.php
```

## 注意事项

- 表名必须使用模块名前缀
- 中间表也加前缀：`mdm_company_user`
- 所有表必须有 `timestamps()`
- 软删除表加 `softDeletes()`
